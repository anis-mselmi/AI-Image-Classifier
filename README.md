# Classification d'Images (Fruits)

Projet simple de classification d'images utilisant TensorFlow pour reconnaître pomme, banane et orange. Le notebook crée la structure du projet, génère des images d'exemple, entraîne un petit modèle CNN et permet de tester une image.

## Structure
project/dataset/ (apple, banana, orange)  
project/models/  
project/results/  

## Utilisation
Ouvrir le notebook dans Google Colab et exécuter les cellules.

## Personnalisation
Ajouter des classes, plus d'images ou modifier le modèle.

## Prérequis
TensorFlow et Pillow.
